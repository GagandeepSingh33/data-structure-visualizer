Searching Algorithms

Understand how different search techniques locate a target inside an array.

What is Searching?

Searching algorithms are methods used to find the position of a target value within a collection of elements such as an array or list.

Different searching techniques trade off simplicity, speed, and the requirement that the data be sorted beforehand.

Linear Search

Linear search scans each element one by one from the beginning until it finds the target or reaches the end of the array.

It works on any array (sorted or unsorted) but has time complexity O(n), so it becomes slow for large datasets.

Binary Search

Binary search works on sorted arrays by repeatedly halving the search range and comparing the middle element with the target.

This divide-and-conquer approach runs in O(log n) time, making it much faster than linear search for large sorted collections.
