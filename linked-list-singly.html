What is a Singly Linked List?

A singly linked list is a sequence of nodes where each node stores a value and a pointer to the next node. Nodes are not stored in contiguous memory, so you can only reach a position by traversing from the head one pointer at a time.

List Initialization

Start with an empty list, or create one from comma-separated values. Each value becomes a node linked from left to right.

Linked List Operations

All operations traverse from the head using pointers, not random indices. Insertion or deletion at head can be O(1); other positions require O(n) traversal.
