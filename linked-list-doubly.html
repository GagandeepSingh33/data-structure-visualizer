What is a Doubly Linked List?

A doubly linked list is a sequence of nodes where each node stores a value and two pointers: prev to the previous node and next to the next node. The prev pointer lets you move one step backward in the list, while the next pointer lets you move one step forward, so traversal can go in both directions.

List Initialization

Start with an empty list, or create one from comma-separated values. Each value becomes a node linked from left to right with both prev and next pointers set.
