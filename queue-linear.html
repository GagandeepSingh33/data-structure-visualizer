What is a Linear Queue?

A linear queue is a simple First-In, First-Out (FIFO) data structure. New elements are added using enqueue at the rear, and removed using dequeue from the front. In an array-based implementation, operations like enqueue, dequeue, and peek typically run in constant time using front and rear indices.

Queue Initialization

Start with an empty linear queue, or create one from comma-separated values. The first value in the list becomes the front of the queue.

Linear Queue View

The linear queue is shown as a horizontal row of items. The front is on the left and the rear is on the right. Enqueue adds an item at the rear; dequeue removes from the front, so the earliest item leaves first.
