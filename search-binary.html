What is Binary Search?

Binary search is an efficient searching algorithm that works on sorted arrays by comparing the target with the middle element and discarding half of the search space each step.

If the target is less than the middle element, it searches the left half; if greater, it searches the right half, repeating until the element is found or the range becomes empty.

Time & Space Complexity

Time complexity: best case O(1) when the target is at the middle; worst and average cases O(log n) because the search range is halved at each step.

Space complexity: O(1) extra for iterative implementations, since only a few indices are tracked while searching in-place on the array.
