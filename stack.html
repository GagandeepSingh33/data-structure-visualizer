What is a Stack?

A stack is a linear data structure that stores elements in a Last-In, First-Out (LIFO) order. New elements are added using push and removed using pop only at one end, called the top of the stack. Operations like push, pop, and peek typically run in constant time.

Stack Initialization

Start with an empty stack, or create one from comma-separated values. The last value in the list becomes the top of the stack.

Stack View

The stack is shown as a vertical column of items. The top element is highlighted at the upper end. Push adds an item on top; pop removes from the top, leaving lower elements untouched.
