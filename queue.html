What is a Queue?

A queue is a linear data structure that stores elements in a First-In, First-Out (FIFO) order. New elements are added using enqueue at the rear and removed using dequeue from the front. Operations like enqueue, dequeue, and peek typically run in constant time when implemented with front and rear indices.

Types of Queue

Linear Queue

A linear queue is a basic FIFO structure where elements are inserted at the rear and deleted from the front in a straight sequence, like a normal waiting line.
