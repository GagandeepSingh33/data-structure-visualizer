B+ trees are a variant of B-trees where all records are stored in the leaves and internal nodes contain only keys for navigation, making range queries efficient.

What is a B+ Tree?

A B+ tree keeps all data records in the leaf level and uses internal nodes only as an index. Leaves are usually linked in a chain, which supports fast range scans and ordered iteration over keys.
