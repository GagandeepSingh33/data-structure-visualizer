What is Radix Sort?

Radix Sort is a non-comparison sorting algorithm that sorts integers by processing individual digits from least significant to most significant, using a stable sort like Counting Sort on each digit.

By repeatedly grouping numbers into buckets based on the current digit and recombining them, Radix Sort achieves linear time O(n · d) for n keys with d digits when the base is fixed.

Complexity & Legend

Time complexity: O(n · d) where n is the number of elements and d is the number of digits in the largest number; space complexity: O(n + b) for temporary buckets in base b (typically 10).
