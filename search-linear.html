What is Linear Search?

Linear search is a sequential searching algorithm that starts from one end of the array and compares each element with the target value.

If a match is found the search stops and returns the index; if the end is reached without a match, the element is reported as not found.

Time & Space Complexity

Time complexity: best case O(1) when the target is at the first position; worst and average cases O(n) because each element may need to be checked.

Space complexity: O(1) extra, since the search operates directly on the existing array without additional data structures.
